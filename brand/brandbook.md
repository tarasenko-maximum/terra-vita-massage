## 1. О бренде

**Terra Vita Massage & Spa** — премиальный массажный салон в Нови-Саде (Сербия).  
Название переводится с латыни как «Земля Жизни». Бренд объединяет натуральные природные элементы (зелень, дерево, земля) с современным спа-комфортом.

**Миссия:** Восстановление гармонии тела и духа через профессиональный массаж и спа-ритуалы.

**Ценности:**
- Натуральность и экологичность
- Профессионализм мастеров
- Индивидуальный подход
- Тёплая, уютная атмосфера

## 2. Логотип

### 2.1 Варианты логотипа

| № | Тип | Файл | Описание |
|---|-----|------|----------|
| 1 | Знаковый (Leaf-T) | `logo-variant-1-leaf-T.png` | Элегантная зелёная ветвь/лист, образующая букву T. Золотой акцент. |
| 2 | Монограмма (TV) | `logo-variant-2-monogram-TV.png` | Переплетающиеся линии T и V в зелёно-золотых тонах. |
| 3 | Символический (Hands) | `logo-variant-3-hands-touch.png` | Абстрактное изображение рук/касания в зелёно-золотой палитре. |

**Рекомендуемый основной логотип:** Вариант 1 (Leaf-T) — наиболее узнаваем, передаёт связь с природой и премиальность.

### 2.2 Правила использования логотипа

#### Охранное поле (clear space)
- Минимальное охранное поле вокруг логотипа = высоте буквы «T» в логотипе.
- Никакие другие элементы (текст, изображения, рамки) не должны вторгаться в это поле.

```
┌─────────────────────────────────┐
│  ← охранное поле = высота "T" → │
│                                 │
│          [ЛОГОТИП]              │
│                                 │
└─────────────────────────────────┘
```

#### Минимальный размер
- **Печать:** ширина логотипа не менее 25 мм.
- **Цифровые носители:** ширина логотипа не менее 120 px.
- Для экранов с высокой плотностью (Retina) использовать @2x версию.

#### Цветовые версии
| Версия | Когда использовать |
|--------|-------------------|
| Полноцветная (зелёный + золото) | Основное применение — белый или светлый фон |
| Монохромная (белая) | Тёмный фон, фотоизображения, футер |
| Монохромная (чёрная) | Ч/б печать, документы |

### 2.3 Чего НЕ делать с логотипом

❌ **Не деформировать** — не растягивать, не сжимать, не искажать пропорции.  
❌ **Не менять цвета** — только из утверждённой палитры.  
❌ **Не добавлять эффекты** — без теней, градиентов, обводок, свечения.  
❌ **Не поворачивать** — логотип всегда в горизонтальном положении.  
❌ **Не размещать на пёстром фоне** — только однородный фон.  
❌ **Не изменять прозрачность** — логотип всегда 100% opacity.  
❌ **Не перекрывать другими элементами.**

## 7. Примеры правильного и неправильного использования

### ✅ Правильно

```
Правильное охранное поле:
┌──────────────────────────────────────┐
│                                      │
│            [ЛОГОТИП]                │
│                                      │
│  Достаточный отступ от краёв         │
│  Фон — однородный светлый            │
└──────────────────────────────────────┘

Правильное цветовое решение:
[ЛОГОТИП] — полноцветный на #F7F5F0
[ЛОГОТИП] — белый на #2E4A3B
```

### ❌ Неправильно

```
❌ Растянутый логотип:
[  Л  О  Г  О  Т  И  П  ]  ← искажённые пропорции

❌ Логотип на пёстром фоне:
┌─────────────────────────┐
│  🌸 🌿 🌸 🌿            │
│    [ЛОГОТИП]            │  ← плохая читаемость
│  🌸 🌿 🌸 🌿            │
└─────────────────────────┘

❌ Цветовая аномалия:
[ЛОГОТИП] — ярко-розовый логотип  ← НЕ из палитры
[ЛОГОТИП] — с тенью и градиентом  ← лишние эффекты

❌ Плохая типографика:
Helvetica вместо Playfair Display  ← не тот шрифт
Размер H1 = 24px                  ← слишком мелко для заголовка
```
